# Data Dictionary

- customers: synthetic customer profile data.
- products: synthetic product catalog data.
- orders: order header data.
- order_items: order line item data.
- web_sessions: synthetic web behavior data.
- marketing_campaigns: campaign metadata.
- email_events: email engagement events.
- support_tickets: customer support records.
- customer_analytics: gold customer-level analytics table.
- marketing_performance: gold campaign performance table.
